Generated additive VFX sprites for the replay motion layer.

These are black-background PNGs intended for `mix-blend-mode: screen`.
The source images were generated with the built-in image generation tool and
copied from `$CODEX_HOME/generated_images` into this project.

- `impact-burst.png`: radial hit burst and shockwave sprite.
- `energy-orb.png`: projectile core and spark sprite.
- `energy-streak.png`: draw trail, beam, and glint sprite.

Prompt pattern:

```text
Use case: stylized-concept
Asset type: additive game VFX sprite for a Pokemon-style card replay web UI
Scene/backdrop: perfectly uniform pure #000000 black background for screen-blend compositing.
Style/medium: polished 2D game VFX sprite, high contrast, clean arcade/anime battle effect.
Constraints: black background must be flat and uniform; no text, watermark, border, grid, UI, or characters.
```
